## Задание

### a)
Разработать абстрактный тип данных — одномерный массив.

Перегруженные операции:
- `[]` — доступ по индексу
- `==` — проверка на равенство
- `!=` — проверка на неравенство

### b)
Разработать иерархию классов:
- 2D Vector
- 3D Vector

Требования:
- приватные поля
- статический член для подсчета максимальной длины
- минимум 2 виртуальные функции

### c)
Создать шаблонный класс одномерного массива:
- массива векторов
- массива целых чисел

Продемонстрировать работу всех операций.

# Структура программы

## Базовый класс `Vector`

Содержит:
- виртуальную функцию вычисления длины
- виртуальную функцию вывода
- статическую переменную максимальной длины

---

## Класс `Vector2D`

Поля:
- `x`
- `y`

Функции:
- вычисление длины 2D вектора
- вывод информации

---

## Класс `Vector3D`

Поля:
- `x`
- `y`
- `z`

Функции:
- вычисление длины 3D вектора
- вывод информации

---

## Шаблонный класс `Array<T>`

Поддерживает:
- динамический массив
- оператор `[]`
- оператор `==`
- оператор `!=`

---
